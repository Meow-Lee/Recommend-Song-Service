# Recommend-Song-Service
Streamlit과 LCEL을 사용해 노래를 추천해주는 챗봇 서비스를 구현하였습니다.

# Requirements
- Python 3 버전 이상
- Groq API 키(console.groq.com)

# Install
1. 가상 환경을 만들고 활성화
```bash
python -m venv .venv
source .venv/Scripts/activate
```

2. 필요한 패키지 설치
```bash
pip install -r requirements.txt
```
```bash
pip install langchain langchain-groq streamlit
```
